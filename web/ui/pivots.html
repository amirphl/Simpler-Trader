Pivot Structure Lab

Lightweight Charts · Binance or CSV candles
Ref / R: reference candle where a pivot candidate starts.
Pivot / P: confirmed swing pivot before the trigger candle.
Target: trigger candle that breaks the candidate level.
Stacked labels: one candle can show multiple marks, displayed as separate lines.
Restart on invalidation: re-scan from j+1 instead of jumping to the invalidating candle — fills structural gaps.
Min swing %: discard pivots whose (high−low)/mid is below this threshold; e.g. 0.5 removes micro-swings.
Structural left bound: use the previous confirmed same-type pivot as the left boundary for the invalidation zone instead of the nearest raw same-direction candle. May widen or narrow the zone depending on data.
Include ref candle: include the reference candle j itself in the argmax/argmin pivot search so the structural extreme is never missed when j holds the highest high or lowest low. Enabled by default.
