Pivot Structure Lab v2

v1 Lightweight Charts · Binance or CSV candles
Bullish candle: finds previous valid bullish candle (separated by ≥1 bearish), scans forward — if range low taken out first, bearish pivot is at argmax high.
Bearish candle: reverse — finds previous valid bearish candle (separated by ≥1 bullish), scans forward — if range high taken out first, bullish pivot is at argmin low.
Candle labels: shown when ≤ threshold candles are visible. Top line = candle index, bottom line = pivot index (or N/A).
Bear pivot: yellow/amber triangle above the pivot candle's high.
Bull pivot: green triangle below the pivot candle's low.
Min swing %: discard pivots whose range (high−low)/mid is below this threshold.
